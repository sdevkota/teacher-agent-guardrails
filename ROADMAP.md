# Roadmap

## 1.0.0

- Dependency-free validator and CLI.
- Domain-specific sample packet.
- Documentation, contribution guide, MIT license, security policy, and GitHub Pages site.

## 1.1.0

- JSON Schema for packet validation.
- More fixtures from safe public examples.
- Machine-readable report format stability tests.

## 1.2.0

- First adapter contributed by the community.
- CI matrix with edge-case packets.
- Better scoring transparency and docs.

## Community tracks

- FERPA/COPPA policy packs
- teacher dashboards
- bias evals
- accommodation templates
