Teacher Agent Guardrails

Guardrails for classroom AI agents that protect students and support teachers.

Contribute on GitHub Run v1.0.0 locally
JavaScript MIT licensed Offline by default Community extensible

Purpose

Education agents can personalize learning, but they also risk privacy leakage, over-surveillance, biased feedback, and replacing teacher judgment.

A classroom deployment checker for student data, teacher oversight, age appropriateness, accommodations, and escalation workflows.

What it does

Validates a domain-specific AI governance packet, scores readiness, and returns concrete findings that contributors can improve.

Why it matters

AI systems are moving from chat into action. This project makes one hard operational risk easier to inspect, test, and govern in public.

Who should use it

Guardrails for classroom AI agents that protect students and support teachers. Builders can start with the CLI, then add adapters, fixtures, schemas, and integrations.

Quick Start

npm test
npm start -- sample

Example Packet

{
  "classroom": {
    "gradeBand": "6-8",
    "subject": "math"
  },
  "privacy": {
    "storesStudentData": true,
    "retentionDays": 30,
    "parentNotice": true
  },
  "oversight": {
    "teacherCanOverride": true,
    "crisisEscalation": true
  }
}
